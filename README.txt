FB Phone Extractor – Unlimited Version

Instructions:
1. Install the extension:
   - Chrome → Extensions → Load unpacked → Select this folder

2. Usage:
   - Go to any Facebook Page
   - Click "Start Extraction"
   - Wait for collection
   - Click "Download CSV" to get phone numbers

3. License System:
   - You need a valid license key to use extraction
   - Without license, buttons are disabled → extension useless
   - Manual Bkash payment → you issue license key
   - Later can implement automated license check system

4. Notes:
   - Works on any PC
   - Unlimited extraction
   - Make sure page fully loaded for best results
